# 3D

## Vorbereitung
<hr/>

- Alle Becher (10 pro Team) müssen vor Spielbeginn in einer <Highlight>3D</Highlight>-Pyramidenformation aufgestellt werden (mehr dazu unter [Becherformation](#becherformation)).

### Becherformation

- Alle Becher müssen in einer <Highlight>3D</Highlight>-Pyramidenformation aufgestellt werden. Dabei muss die Spitze der Pyramide zur anderen Tischseite zeigen und es müssen sich alle Ränder der Becher berühren.
- <Highlight>Die erste Ebene der Pyramide besteht aus sechs Bechern. Die zweite Ebene besteht aus drei Bechern und wird in die Mitte der sechs Becher gestellt. Der letzte 10. Becher wird in die Mitte der drei Becher der zweiten Ebene gestellt.</Highlight>

## Allgemeine Regeln
<hr/>

### Allgemeines

#### Art 2 - Entfernen der Becher

5. <Highlight>Wird ein Becher der obersten Ebene getroffen, so wird genau dieser Becher entfernt.</Highlight>
6. <Highlight>Wird ein Becher einer Ebene getroffen, die nicht die oberste ist, so wird ein Becher der obersten Ebene entfernt. Das Team, dem die Becher gehören, darf sich aussuchen, welcher Becher entfernt wird.</Highlight>

## Sonstiges
<hr/>

### Spezialwurfkombinationen

- <Highlight>1. Treffer im Spiel = oberster Becher &rarr; Oberster Becher + 2 weitere</Highlight>
- <Highlight>1. Treffer im Spiel = oberster Becher + jegliche Kombination &rarr; Oberster Becher + 2 weitere + andere Kombination</Highlight>

## Kombinationsmöglichkeiten
<hr/>

Dieser Flavor ist ohne weitere Einschränkungen mit allen anderen Flavors kombinierbar.
